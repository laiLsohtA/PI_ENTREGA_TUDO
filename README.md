# EntregaTudo

Projeto Integrador desenvolvido no curso de Engenharia de Software da UniEVANGÉLICA.

## Sobre o projeto

O EntregaTudo é uma aplicação web voltada para marketplace e delivery, permitindo a navegação por produtos, filtragem por categorias, visualização de detalhes, cadastro de produtos e autenticação de usuários.

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router
- Zod
- GitHub

## Funcionalidades implementadas

- Listagem de produtos
- Filtro por categoria
- Busca textual
- Visualização de detalhes do produto
- Cadastro de produtos
- Login e cadastro de usuários
- Página de perfil autenticado
